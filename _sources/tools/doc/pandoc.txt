Pandoc
======

Pandoc (http://pandoc.org/) can be used to convert from one markup format to
another. It contains a long list of supported formats. Some examples are:

.. prompt:: bash
   
   pandoc input.txt -o output.html 
   pandoc page.hml -o output.rst


Her the firt sfile is the input and the file after the `-o` is the
output file. The conversion identifies the ending and if available
converts the file.
