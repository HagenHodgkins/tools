2.4.1. Git¶

  1. Create an account on github.com
  2. notify me about your accountant on it.
  3. Once you have done this you need to know about how to use git. You can read the git manual.

Here a summary of the commands that are important for you:

git clone
git commit -m
git commit -a
git push
git pull

The first thing you want to do is to set your git username and e-mail
